# Awesome Dotties

## Dependencies

- AwesomeWM
  - picom
  - nm-applet
  - protonvpn-cli
- VIM
- zsh

## Setup

```bash
./setup
```

This script will:

- Install Vundle for vim plugins, then run :PluginInstall
- Install stow, rg, and py-wal if they aren't already installed
- Install oh-my-zsh if it's not already installed
- Stow all config files
